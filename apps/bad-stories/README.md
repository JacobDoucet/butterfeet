# Bad Stories

Bad Stories is a local-only Android app for creating goofy fill-in-the-blank stories.

## Project Areas

- `android/`: Kotlin + Compose app
- `content/`: source JSON story packs
- `docs/`: product notes and future plans

## Product Principles

- Keep it funny and fast
- Keep architecture boring and easy to evolve
- Avoid backend complexity until it is clearly needed
